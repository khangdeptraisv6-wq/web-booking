# ✍️ HƯỚNG DẪN LÀM BÀI - NHÓM 11
Chào anh em, đây là các bước để chạy được web đặt lịch này trên máy cá nhân:

## 1️⃣ Chuẩn bị môi trường
* Cài đặt **Node.js** bản mới nhất.
* Cài đặt **SQL Server** để làm Database.

## 2️⃣ Cách chạy Code
* Mở Terminal trong thư mục này, gõ lệnh: `npm install` để cài thư viện.
* Chạy file SQL (tao có gửi kèm) để tạo bảng dữ liệu.
* Gõ lệnh: `node index.js` để bắt đầu chạy web.

## 3️⃣ Vào web
* Mở trình duyệt gõ: `localhost:3000`
* Đăng ký tài khoản mới rồi đăng nhập là xong!
